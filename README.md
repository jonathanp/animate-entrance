# animate-entrance

[![NPM version][npm-image]][npm-url]

> A jQuery plugin for animating the entrances of DOM elements as they appear in the viewport.


## Install

```
$ npm install --save animate-entrance
```


## Usage

```js
$('.selector').animateEntrance({
    className: 'is-entered',
    threshold: 0.5,
    throttle: 100
});
```
